# E-commerce Backend Setup

You are an expert in building MERN stack applications. Help me set up the backend for an e-commerce website selling home appliances.

The frontend is already built with React and uses Firebase for auth, but now we need to switch to a full MERN stack with MongoDB.

Generate complete, runnable code files for the backend using Express.js, Node.js, and MongoDB.

When asked to create something, provide the full file content, including all necessary imports, and ensure it's ready to run.

Key features needed:
- User authentication (register, login, logout) using JWT
- Product management (CRUD for products)
- Cart functionality
- Order management
- Admin dashboard for managing products, orders, users

Models: User, Product, Order, Cart, Category

Start with APIs for user registration/login, then product listing and management.

Use Mongoose for MongoDB models.
Include proper error handling, validation (express-validator), and security (bcrypt for password hashing, JWT for auth).
